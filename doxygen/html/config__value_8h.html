◆ CAN_MODE

enum CAN_MODE

Modes d'utilisation CAN (envoi/reception).

Enumerator
RECEVE_ONLY 

Reception seule: les updates entrantes peuvent modifier la valeur; l'ecriture locale est bloquee.

SEND_ONLY 

Emission seule: la valeur locale est emise, les updates entrantes ne la modifient pas.

SEND_AND_RECEVE 

Emission + reception: ecriture locale et updates entrantes toutes deux autorisees.
